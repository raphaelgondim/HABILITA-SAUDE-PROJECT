# Habilita Saúde

## Sobre o Projeto

O Habilita Saúde é uma plataforma de cursos online voltada para profissionais e estudantes da área da saúde.

O projeto foi desenvolvido durante a Residência Porto Digital com o objetivo de criar um ambiente digital intuitivo para gerenciamento e acesso a cursos especializados.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Figma
- GitHub

## Funcionalidades

### Área do Professor
- Dashboard
- Meus Cursos
- Criar Curso
- Relatórios
- Usuários
- Notificações
- Suporte

### Área do Aluno
- Visualização de Cursos
- Acompanhamento de Progresso
- Certificados
- Notificações

## Protótipo

O projeto foi inicialmente desenvolvido no Figma e posteriormente implementado utilizando HTML, CSS e JavaScript.
